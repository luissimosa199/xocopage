# Xoco
## E-commerce web

Prototipo de web e-comerce para emprendimiento de tabletas y bombones de chocolate.

Herramientas:
- HTML, CSS y JS

## Características

- Galería de fotos.
- Lista de productos.
- Carrito de compras que permite agregar y borrar items, calcular el total de la compra, vaciar el carrito o proceder con la orden de compra.
- Sección "Sobre Nosotros".
- Sección "Footer" con los enlaces a las redes del emprendimiento.
- Al proceder con la compra, la API de Whatsapp nos permite enviar la orden de compra con los datos del cliente directamente a los emprendedores.
